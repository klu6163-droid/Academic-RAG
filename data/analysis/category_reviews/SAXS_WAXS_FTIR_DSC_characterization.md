## 1. Research Background

SAXS/WAXS/FTIR/DSC是聚合物微结构表征的核心技术组合。SAXS提供纳米尺度结构信息（长周期、domain size、界面）；WAXS/XRD提供晶体结构和结晶度；FTIR揭示化学键状态和氢键相互作用；DSC测量热转变（Tg、Tm、Tc、ΔH）。这些表征手段对理解PU微相分离、结晶行为、结构-性能关系至关重要。

## 3. Core Mechanisms / Characterization Techniques

**SAXS Analysis Methods:**
- Lorentz-corrected I(q) vs q plots
- Long period from peak position: L = 2π/q*
- Domain size, interface thickness from correlation function
- FWHM → size distribution / disorder

**FTIR Characterization:**
- Hydrogen bonding index (HBI) from N-H / C=O peak shifts
- Phase separation index from hard/soft segment absorbance ratios
- Peak deconvolution for ordered vs disordered domains

**DSC Analysis:**
- Tg (soft segment), Tm (hard/soft segment), Tc
- Crystallinity from ΔHf
- Multiple endotherms → microphase mixing
