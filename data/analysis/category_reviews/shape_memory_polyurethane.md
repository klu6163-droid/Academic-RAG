## 1. Research Background

Shape memory polyurethane (SMPU) can fix a temporary shape and recover to its permanent shape upon external stimulus (heat, moisture, light). The shape memory effect originates from the phase-separated morphology where hard segments act as physical crosslinks and soft segments serve as switching domains.

## 3. Core Dynamic Bond / Physical Interaction Mechanisms

**Shape Memory Mechanisms:**
- Thermally-induced shape memory (Tg or Tm of soft segment)
- Moisture-induced shape memory
- Two-way shape memory (reversible actuation)
- Triple/multi-shape memory
